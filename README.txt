1. To edit the instructions that will be executed edit the Test.txt File.

2. To run our application double click the RUN.bat file on a computer with windows 7 or greater. 

3. To look at our code you can open the files in any text editor. 


If all else fails you can run the executable jar file by shift right clicking in the file browser where the project is located. 
Click on windows powershell or command window. 

Type the following command: java -jar Test.jar
The window should appear and you should be able to see the output. 

The use of -1 is reserved for almost all of our flags. It should not be used as a value.
